Lexserver

Look up words:

words=hund,häst

http://localhost:8787/lexlookup?lexicons=sv.se.nst&words=hund,häst

Look up words, pretty print:

words=hund,häst&pp=yes

http://localhost:8787/lexlookup?lexicons=sv.se.ns&words=hund,häst&pp=yes

(Value of 'pp' parameter does not matter, as long there is one.)

transcriptionlike

Look up word dom where transcription includes string o:

http://localhost:8787/lexlookup?lexicons=sv.se.ns&words=dom&transcriptionlike=%25o:%25&pp=yes

(For this to work properly, the Sqlite db must be instructed that PRAGMA case_sensitive_like=ON)
