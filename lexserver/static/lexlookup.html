Lexserver

Look up words:

words=hund,häst

/lexlookup?lexicons=sv-se.nst&words=hund,häst

transcriptionlike

Look up word dom where transcription includes string o:

/lexlookup?lexicons=sv-se.nst&words=dom&transcriptionlike=%25o:%25&pp=yes

(For this to work properly, the Sqlite db must be instructed that PRAGMA case_sensitive_like=ON)

Look up lemmas

/lexlookup?lemmas=spring&pp=y

Add entry

/addentry?lexicon=LEX_NAME&entry={...}

where {...} is a JSON entry.

Update entry

/updateentry?entry={...}

where {...} is a JSON entry.
