Create db and start server

github.com/stts-se/pronlex/lexserver$ createEmptyDB pronlex.db
github.com/stts-se/pronlex/lexserver$ addNSTLexToDB sv.se.nst pronlex.db [PATH TO NST LEX]/swe030224NST.pron_utf8.txt 
github.com/stts-se/pronlex/lexserver$ CompileDaemon -command lexserver

Lexserver

List lexicons:

http://localhost:8787/listlexicons

List lexicons, pretty print:

http://localhost:8787/listlexicons?pp=yes

(Value of 'pp' parameter does not matter, as long there is one.)

Lexicon look up:

http://localhost:8787/lexlookup
